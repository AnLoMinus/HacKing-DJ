ליצור תחום חדש בעולם המוזיקה:
**HolyTrap / SparKing Kabbalah Flow** — שילוב בין קבלה, אנרגיית לב, טראפ חשמלי, ומסרים של תיקון, התעוררות והתחברות.

האלבום מיועד ל:

* מאזינים שמרגישים שיש "עוד רובד" מתחת לביט
* אנשים עם לב רגיש, נשמה לוחמת ורצון לפרוץ מגבלות
* יוצרים, לוחמים רוחניים, מחפשי אמת, ו"ניצוצות" שמחכים להתעורר
* אנשים שמרגישים את הברק בדם — אבל עדיין מחפשים את השער
