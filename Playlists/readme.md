## 🔄 איך זה עובד בפועל (Flow פשוט וברור)

### 🎵 העלאת שיר

1. אתה גורר קובץ `.mp3` אל
   `Playlists/שם-הפלייליסט/tracks/`
2. עושה **Commit**

### 🤖 אוטומציה (GitHub Actions)

* נסרקת התיקייה
* נבנה `manifest.json` לפלייליסט
* מתעדכן `playlists.json` הכללי
