## 📝 איך לתרום?

אנחנו תמיד שמחים לקבל תרומות! ראה את [CONTRIBUTING.md](CONTRIBUTING.md) לפרטים.

### דרכים לתרום:

1. **דיווח על באגים** 🐛
   - פתח Issue עם תיאור מפורט
   - כלול צעדים לשחזור

2. **הצעת תכונות** 💡
   - פתח Issue עם תיאור התכונה
   - הסבר למה זה יעזור

3. **שיפור קוד** 💻
   - Fork את הפרויקט
   - צור Branch חדש
   - שלח Pull Request

4. **שיפור תיעוד** 📚
   - תיקון שגיאות כתיב
   - הוספת דוגמאות
   - שיפור הסברים

5. **עיצוב ו-UX** 🎨
   - שיפורי ממשק
   - אנימציות
   - Responsive Design

6. **תרגום** 🌍
   - תרגום למשתמשים ברחבי העולם
   - שיפור נגישות
